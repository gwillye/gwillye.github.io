🎬 Meu Catálogo de Filmes

Um catálogo estilo Filmow que roda 100% no navegador. Busca uma API pública gratuita de filmes (TMDB) e organiza tudo em quatro listas — já vi · quero ver · quero rever · favoritos. Suas listas ficam salvas no seu navegador (localStorage).

A busca usa a TMDB API (chave gratuita). A key fica só no seu navegador. Sem key, suas listas continuam funcionando — só a busca online fica desabilitada.
Digite uma busca, ou abra uma das suas listas.
Client-side demo · TMDB as the free public API · your lists stay in your browser · part of @gwillye's portfolio
